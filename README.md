# harmony eth bridge frontend
Ethereum<>Harmony two way bridge (trusted version)



## Install instructions

### Requirements 

* nodejs 

### Commands

* Fetch repo 

```
git clone git@github.com:harmony-one/ethhmy-bridge.frontend.git
```

* Install dependencies

```
npm install
```

* Develop

```
npm run dev
```

* Build

```
npm run build
```

* Start

```
cd build 
serve
```
